# Security Policy

## Supported Versions

Please carefully check the version of the project you are using to know if it is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of the AI Invoice Processing Agent seriously. If you discover a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

To report a vulnerability:
1. **GitHub Private Vulnerability Reporting**: This repository has Private Vulnerability Reporting enabled. You can privately report a security issue by going to the **Security** tab of this repository, clicking on **Advisories**, and selecting **Report a vulnerability**. This ensures the issue is kept confidential while we work on a fix.
2. We strictly adhere to coordinated vulnerability disclosure. Please do not create public issues or mention the vulnerability in public forums until it has been patched.

We will acknowledge receipt of your vulnerability report within 72 hours and provide updates on the resolution.
