# Contributing to AI Invoice Processing Agent

Thank you for your interest in contributing! We welcome all contributions to make this AI agent even better.

## How to Contribute

1. **Fork the repository**.
2. **Create a local branch** for your feature or bug fix: `git checkout -b feature/your-awesome-feature`.
3. **Commit your changes**: `git commit -m "Add some awesome feature"`.
4. **Push to your branch**: `git push origin feature/your-awesome-feature`.
5. **Open a Pull Request**.

## Environment Setup

Ensure you have both the `frontend` and `backend` environments set up as described in the [README.md](./README.md).

## Code Style

- **Frontend**: Follow the standard Next.js / TypeScript patterns. Use `eslint` for linting.
- **Backend**: Follow PEP 8 for Python code.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the GitHub repository.
